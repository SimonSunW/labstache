VI and Control Reference

VIs and classes for create custom partial loaders.

Palette Object Description
Data A partial loader that saves the partial templates as either a string or path.
File A partial loader that saves the partial templates in a file that is read at render-time and the key is the name of the file without the extension.
String A partial loader that save the partial templates as a LabVIEW string and the key is identifier of the string value in a map.
