Read Value VI

Owning Palette: Boolean Classes, Controls, and VIs

Requires: Labstache

Gets the internal, LabVIEW boolean value for the Mustache boolean value.

Boolean In represents a Mustache boolean value using a LabVIEW boolean (true or false).
The Error In describes error conditions that occur before this node runs. This input provides standard error in functionality.

Right-click the Error In indicator on the front panel and select Explain Error or Explain Warning from the shortcut menu for more information about the error.
The Boolean Out is the same object from Boolean In.
The Value is the internal LabVIEW boolean.
The Error Out contains error information. This provides standard error out functionality.

Right-click the Error Out indicator on the front panel and select Explain Error or Explain Warning from the shortcut menu for more information about the error.
