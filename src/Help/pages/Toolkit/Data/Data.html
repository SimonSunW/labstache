VI and Control Reference

Use these VIs to wrap LabVIEW values into Mustache data to control the rendering of templates.

Palette Object Description
Array Creates a new Array datum.
Boolean Creates a new Boolean datum.
Datum The abstract base class for all data types.
Function Executes a lambda, or anonymous function.
Map Creates a new Map datum.
Null Creates a new Null, or none, datum.
Path Creates a new Path datum.
String Creates a new String datum.
