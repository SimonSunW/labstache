Null Controls and VIs

Owning Palette: Data Classes, Controls, and VIs

Requires: Labstache

The Null class represents a nothing, none, or NULL Mustache data type and value. A Null object should always be considered FALSE in truthiness and equality testing of Mustache data types and values.

Palette Object Description
New Creates a new Mustache NULL data type object. Generally, if the Mustache data is NULL, then Mustache template tag will not be rendered.
Read Type Get the Mustache data type of an object.
