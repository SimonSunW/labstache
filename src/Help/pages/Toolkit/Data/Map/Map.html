Map Controls and VIs

Owning Palette: Data Classes, Controls, and VIs

Requires: Labstache

The Map class represents a collection of key-value pairs of Mustache data types and values.

Palette Object Description
Find Retrieves a Mustache data value based on the key. Keys can only be strings.
From Wraps a LabVIEW collection of key-value pairs into a Mustache data type object.
Read Type Get the Mustache data type of an object.
