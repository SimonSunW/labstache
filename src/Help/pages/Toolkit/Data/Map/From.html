From VI

Owning Palette: Map Classes, Controls, and VIs

Requires: Labstache

Wraps a LabVIEW collection of key-value pairs into a Mustache data type object.

Boolean

The Key is the LabVIEW string to use to uniquely identify the value within the collection of key-value pairs.
Map represents a Mustache data type with a LabVIEW type and value.

Datum

The Value is a LabVIEW value that has already been wrapped, or converted, to a Mustache data value. A key-value pair will be created with this as the value.
The Key is the LabVIEW string to use to uniquely identify the value within the collection of key-value pairs.
Map represents a Mustache data type with a LabVIEW type and value.

Path

The Value is the LabVIEW path that should be wrapped, or converted, to a Mustache data value.
The Key is the LabVIEW string to use to uniquely identify the value within the collection of key-value pairs.
Map represents a Mustache data type with a LabVIEW type and value.

String

The Key is the LabVIEW string to use to uniquely identify the value within the collection of key-value pairs.
The Value is the LabVIEW string that should be wrapped, or converted, to a Mustache data value.
Map represents a Mustache data type with a LabVIEW type and value.

Map Array

Map represents a Mustache data type with a LabVIEW type and value.
