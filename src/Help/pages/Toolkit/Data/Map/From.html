From VI

Owning Palette: Map Classes, Controls, and VIs

Requires: Labstache

Wraps, or converts, a LabVIEW value into a Mustache array of key-value pairs.

Boolean

Value is the LabVIEW boolean value to wrap as a Mustache boolean value.
Key is the LabVIEW string to use to uniquely identify the value within the collection of key-value pairs.
Map represents a Mustache data type with a LabVIEW type and value.

Datum

Value is a LabVIEW value that has already been wrapped, or converted, to a Mustache data value. A key-value pair will be created with this as the value.
Key is the LabVIEW string to use to uniquely identify the value within the collection of key-value pairs.
Map represents a Mustache data type with a LabVIEW type and value.

Path

Value is the LabVIEW path that should be wrapped, or converted, to a Mustache data value.
Key is the LabVIEW string to use to uniquely identify the value within the collection of key-value pairs.
Map represents a Mustache data type with a LabVIEW type and value.

String

Key is the LabVIEW string to use to uniquely identify the value within the collection of key-value pairs.
Value is the LabVIEW string that should be wrapped, or converted, to a Mustache data value.
Map represents a Mustache data type with a LabVIEW type and value.

Pair Array

Data is an array of key-value pairs that is the LabVIEW values represented as a Mustache Map datum.
Map represents a Mustache data type with a LabVIEW type and value.

Pair

Map represents a Mustache data type with a LabVIEW type and value.

String Array

Map represents a Mustache data type with a LabVIEW type and value.
