Find VI

Owning Palette: Map Classes, Controls, and VIs

Requires: Labstache

Retrieves a Mustache data value based on the key. Keys can only be strings.

Array In represents a Mustache map value using a LabVIEW array of key-value pair clusters.
The Error In describes error conditions that occur before this node runs. This input provides standard error in functionality.

Right-click the Error In indicator on the front panel and select Explain Error or Explain Warning from the shortcut menu for more information about the error.
The Key is a string that uniquely identifies a Mustache data value in the collection of key-value pairs.
The Value is the Mustache data value associated with the Key. If a key-value pair matching the Key is not found, then this will be a Null object.
The Map Out is the same object from Map In.
The Found? is a boolean value indicating if a key-value pair matching the Key was found (TRUE) or not (FALSE).
The Error Out contains error information. This provides standard error out functionality.

Right-click the Error Out indicator on the front panel and select Explain Error or Explain Warning from the shortcut menu for more information about the error.
