Datum Controls and VIs

Owning Palette: Data Classes, Controls, and VIs

Requires: Labstache

The Datum class is the abstract base class for all data types that are recognized by the compiler and used to control the behavior of the rendering process. These are the data types that are recognized by a Mustache template and are largely identical to the JavaScript/JSON data types and primitatives. Objects of this class should not be directly created. Instead, a new class that inherints from the Datum class should be created.
