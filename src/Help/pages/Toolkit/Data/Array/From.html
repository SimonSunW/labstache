From VI

Owning Palette: Array Classes, Controls, and VIs

Requires: Labstache

Wraps an array of LabVIEW primitive values into an array of Mustache data type objects.

Data

Data is an array of Mustache data objects that have already been wrapped, or converted, into a Mustache data value.
Array represents a Mustache data type with a LabVIEW type and value.

Strings

Strings is an array of LabVIEW strings that should be wrapped, or converted, into a Mustache data array. Each element will be converted to a String Mustache data value.
Array represents a Mustache data type with a LabVIEW type and value.

Paths

Paths is an array of LabVIEW paths that should be wrapped, or converted, into a Mustache data array. Each element will be converted to a Path Mustache data value.
Array represents a Mustache data type with a LabVIEW type and value.

Mapped Data

Data is an array of Mustache data objects that have already been wrapped, or converted, into a Mustache data value. Each element will be converted to a key-value pair with the key from Key.
The Key is the same used to create the key-value pairs from Strings. This is useful when working with sections that loop should loop over data.
Array represents a Mustache data type with a LabVIEW type and value.

Mapped String

Strings is an array of LabVIEW strings that should be wrapped, or converted, into a Mustache data array. Each element will be converted to a key-value pair with the key from Key and the value converted to a String Mustache data value.
The Key is the same used to create the key-value pairs from Strings. This is useful when working with sections that loop should loop over data.
Array represents a Mustache data type with a LabVIEW type and value.

Mapped Paths

Paths is an array of LabVIEW strings that should be wrapped, or converted, into a Mustache data array. Each element will be converted to a key-value pair with the key from Key and the value converted to a Path Mustache data value.
The Key is the same used to create the key-value pairs from Strings. This is useful when working with sections that loop should loop over data.
Array represents a Mustache data type with a LabVIEW type and value.
