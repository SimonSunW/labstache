Execute VI

Owning Palette: Function Classes, Controls, and VIs

Requires: Labstache

Renders a block from a raw Mustache template at render time.

Function In represents a callable object, such as a function or lambda. A lambda is also known as an anonymous function.
The Error In describes error conditions that occur before this node runs. This input provides standard error in functionality.

Right-click the Error In indicator on the front panel and select Explain Error or Explain Warning from the shortcut menu for more information about the error.
The Source is a block of raw, unrendered Mustache template to be rendered by the implementation of this VI.
The Nodes array represents an abstract syntax tree of the rendered Source based on the implementation of this callable object (function or lambda).
The Function Out object is the same object as the Function In control.
The Error Out contains error information. This provides standard error out functionality.

Right-click the Error Out indicator on the front panel and select Explain Error or Explain Warning from the shortcut menu for more information about the error.
