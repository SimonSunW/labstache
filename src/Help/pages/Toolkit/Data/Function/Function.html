Function Controls and VIs

Owning Palette: Data Classes, Controls, and VIs

Requires: Labstache

The Function class represents a Mustache Lambda, or anonymous function, which is a callable object. This class is essentially an abstract class and should only be sub-classed with a concrete implementation. A Mustache Lambda receives the unrendered, raw input and outputs rendered text. This is useful for implementing filters or caching.

Palette Object Description
Execute Renders a block from a raw Mustache template at render time.
Read Type Get the Mustache data type of an object.
